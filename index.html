用HEXO和Github建立个人网站

我的个人网站 https://xiaowangya.github.io/

准备

环境配置

  • Node.js 安装

    根据自己的电脑平台从该下载链接安装。我个人是windows平台，选择的是长期维护版的“64bit.msi”文件。

    双击运行安装文件，安装过程全程下一步，记得勾选 自动加入Path 选项。

    因为Npm是Node.js的包管理工具(package manager)，所以在安装Node.js的时候会自动安装npm.

    检查是否安装成功

      npm -V #检查npm安装
  node -v #检查node.js
  path #检查环境变量

  • Git安装

    见此教程

框架和主题安装

  • HEXO框架安装

    随后打开CMD, 在命令行中输入

      npm install -g hexo-cli

    其他平台（如 MAC）参见link

    安装hexo插件

      npm install hexo-deployer-git --save

    在下载中意的主题，并在对应的github网站处下载。本方案选择的是NEXT主题。

修改个人信息

选择网站主题

首先，创建一个本地用于存储网站的文件夹。然后在该文件夹中，执行

git init
hexo init dir_name #dir_name是你定义的文件夹名字
cd dir_name
npm install

在HEXO中选好主题并安装到themes文件夹下

cd themes
git clone https://github.com/your_theme.git #前面选好的主题对应的git仓库

更新相关信息

用编辑器打开文件夹中的_config.yml文件，并修改

url: https://username.github.io/project

deploy:
    type: 'git'
    repo: https://github.com/XiaoWangya/xiaowangya.github.io
    branch: master
theme: hexo-theme-next
